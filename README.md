Изучение языка происходит по книге "Язык программирования C лекции и упражнения" 6-е издание Стивен Прата.

В данном репозитории будут представлены файлы с выполнеными домашними работами (в дальнейшем возможна реорганизация по папкам). Реализация представляет собой файл с названием "chapter_one.c", где one обозначает номер главы и такой же .exe файл.
